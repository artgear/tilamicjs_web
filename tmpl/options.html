Constructor Options

The number of tiles is represented by the product of the rows and cols.
The more tiles, the more resources. You should keep them within reasonable bounds.

  • rows

    The number of rows.

    default: 3
    type: integer

  • cols

    The number of columns.

    default: 3
    type: integer

  • attrPrefix

    Adding prefix to the html class names tilamic.js uses.

    default: '' (blank string)
    type: string

  • aspectRatio

    This item is required if you make your content with responsive design.
    Setting the area width using a percent value and aspectRatio, the content will be elastic.
    The value = height / width.
    In case you don't need flexibility, set both the area width and height.

    default: null
    type: float

  • isCSS3Supported

    Usually this item is set to the appropriate value automatically.
    If you want fallback action on all browsers, set this false.

    default: true if the browser supports CSS3 features or else false.
    type: boolean
