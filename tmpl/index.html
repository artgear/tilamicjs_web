Features

  • IE7, 8, 9 compatible (Not flip, fallback action)
  • Easy-to-customize flip sequence
  • Responsive

How to install

  • 1. Link jQuery and tilamic.js files. jQuery version must be 1.8.0 or newer. 

    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
<script src="js/tilamic.js"></script>

  • 2. Create HTML like this. 

    <div id="tiles">
  <img src="img/photo1.jpg" alt="photo1" />
  <img src="img/photo2.jpg" alt="photo2" />
  <img src="img/photo3.jpg" alt="photo3" />
  <img src="img/photo4.jpg" alt="photo4" />
</div>

  • 3. Construct a new Tilamic object. First parameter: jQuery selector. Second: options. 

    $(document).ready(function(){
  var tilamic = new Tilamic('#tiles', {
    'rows': 3,
    'cols': 5,
    'aspectRatio': 853/1280
});

  • 3-1. If you need auto-resizing for responsive design, bind 'resize' method to window resize event. 

    $(window).on('resize', function(e){
  tilamic.resize();
});

  • 4. Flip. 

    tilamic.flip(1); // flip all tiles in unison
    tilamic.seqFlip(2); // flip all tiles in sequence

  • note: tilamic.js itself has not any methods for slideshow. If you need it, write like this.

    var count = 1;
setInterval(function(){
  tilamic.flip(count % 4);
  count++
}, 3000);

Customize

To customize flip sequence, use a combination of two methods, "filter" and "setOrigin".
The .filter() method constructs a new Tilamic object that have the tiles 'isActive' status is modified.
The "setOrigin" methods construct a new Tilamic object which have the tiles the CSS3 transform-origin is modified.
More information, see filter and setOrigin. 

tilamic.filter(function(index, col, row){
  return (col + row) % 2 == 0;
}).setOrigin('centerOfAll').flip(3);
