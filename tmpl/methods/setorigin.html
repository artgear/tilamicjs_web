.setOrigin()

.setOrigin(keyword)

keyword
Type: String

Choose among from the following four.
'each'
'centerOfAll'
'centerOfRow'
'middleOfCol'

The .setOrigin() method constructs a new Tilamic object that have the tiles CSS3 transform-origin property is modified.
Passing 'each' keyword, the transform-origin of all of 'isActive' tiles will be set to the center of its own.
Passing 'centerOfAll' keyword, the transform-origin of all of 'isActive' tiles will be set to the center of the area.
Passing 'centerOfRow' keyword, the transform-origin of all of 'isActive' tiles will be set to the center of the row it belongs.
Passing 'middleOfCol' keyword, the transform-origin of all of 'isActive' tiles will be set to the middle of the col it belongs.
Passing something else, .setOrigin() method does not do anything.
'isActive' is the attribute that represents the state of the tile. In the state immediately after construction of Tilamic object, all of its tiles are 'isActive'.For more information about 'isActive' see the .filter() method.

Example 

$(function(){
  var tilamic = new Tilamic('#tiles', {
    'aspectRatio': 853/ 1280
  });

  var imgID = 1;
  var isAxisX = false;
  var originType = ['each', 'centerOfAll', 'centerOfRow', 'middleOfCol'];
  
  setInterval(function(){
    tilamic.setOrigin(originType[imgID % 4]).flip(imgID % 4, {
      'isAxisX': isAxisX
    });
    isAxisX = !isAxisX;
    imgID++;
  }, 2000);
});
