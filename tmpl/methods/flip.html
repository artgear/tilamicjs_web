.flip()

.flip(imgID[,options])

imgID
Type: number or string

The 'imgID' is decided according to the original HTML <img> order and its 'alt' attribute. 

<div id="tiles">
  <img src="img/photo1.jpg" alt="photo1" /> <!-- .flip(0) or .flip('photo1') calls this image -->
  <img src="img/photo2.jpg" alt="photo2" /> <!-- .flip(1) or .flip('photo2') -->
  <img src="img/photo3.jpg" alt="photo3" /> <!-- .flip(2) or .flip('photo3') -->
  <img src="img/photo4.jpg" alt="photo4" /> <!-- .flip(3) or .flip('photo4') -->
</div>
options
Type: object

A map of additional options to pass to the method.

  • isAxisX(default: false)
    Type: boolean

    If this value is true, the axis of rotation of the tile is set to X-axis. If false, Y-axis.

  • isReverse(default: false)
    Type: boolean

    If this value is true, reverse the direction of rotation of the tile.

When the .flip() method is called, all of 'isActive' tiles flip in unison.
'isActive' is the attribute that represents the state of the tile. In the state immediately after construction of Tilamic object, all of its tiles are 'isActive'.For more information about 'isActive' see the .filter() method.

Example 

$(function(){
  var tilamic = new Tilamic('#tiles', {
    'aspectRatio': 853/ 1280
  });

  var isAxisX = true;
  var imgID = 1;
  
  setInterval(function(){
    tilamic.flip(imgID % 4, {
      'isAxisX': isAxisX
    });
    imgID++;
    isAxisX = !isAxisX;
  }, 3000);
});
