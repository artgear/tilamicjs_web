.on()

.on(events[, data], handler)

events
Type: String

One or more space-separated event types.

data
Type: Object

Data to be passed to the handler in jQuery-Event object when an event is triggered.

handler
Type: Function(jQuery-Event eventObject)

A function to execute when the event is triggered.

The .on() method attaches event handlers to the 'isActive' tiles.
This method is like jQuery .on() method.
The data to be passed to the handler includes the information about the tile on which the event is triggered.
Passed the data as 'event', 'event.data["tileIndex"]' is it.
Note: If you want to attaches the same function to all of the tiles, you should attaches it to the area instead of the tiles.

Example 

<script>
$(function(){
  var tilamic = new Tilamic('#tiles', {
    'aspectRatio': 853/ 1280
  });

  tilamic.on('click', function(event){
    window.alert('Number ' + event.data['tileIndex'] + ' tile is clicked!');
  });

  var imgID = 1;
  setInterval(function(){
    tilamic.flip(imgID % 4);
    imgID++;
  }, 3000);
});
</script>

Click some one of the tiles.
